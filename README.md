## Runnig the backend

Navigate to the `~/FullStackChallenge/part9/patientorBE` folder:
```

$ node run dev
```

## Runnig the frontend

### You'll need the `patientorBE` application running

Navigate to the `~/FullStackChallenge/part9/patientor` folder:
```
$ npm install
$ npm start
```
